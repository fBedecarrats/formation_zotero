# Checklist préparation Zotero

## Cadrage
- [ ] Confirmer le format exact de la séance
- [ ] Confirmer la durée utile : 14h00–16h30
- [ ] Fixer 3 objectifs pédagogiques prioritaires
- [ ] Choisir l'exercice fil rouge

## Technique
- [ ] Zotero installé sur la machine de démo
- [ ] Connecteur navigateur testé
- [ ] Word ou LibreOffice testé
- [ ] Accès internet vérifié
- [ ] Plan B prêt en cas d'import défaillant

## Contenu
- [ ] Bibliothèque de démonstration prête
- [ ] 5 à 7 références de test sélectionnées
- [ ] 1 doublon prévu
- [ ] 1 PDF mal reconnu prévu
- [ ] 1 rapport institutionnel prévu
- [ ] 1 référence HAL ou OpenEdition prévue

## Supports
- [ ] Support de 8 à 12 diapositives prêt
- [ ] Fiche exercice prête
- [ ] Fiche mémo prête
- [ ] Document de test pour insertion de citations prêt

## Animation
- [ ] Introduction de 10 minutes prête
- [ ] Démonstration d'import prête
- [ ] Démonstration de nettoyage prête
- [ ] Démonstration de partage prête
- [ ] Démonstration de bibliographie prête
- [ ] Conclusion prête

## Communication
- [ ] Message de préparation envoyé aux participant·es
- [ ] Consigne “venir avec 3 à 5 références” envoyée
- [ ] Liens de téléchargement Zotero envoyés
- [ ] Attentes de l'atelier formulées clairement

## Jour J
- [ ] Ouvrir Zotero avant l'arrivée du groupe
- [ ] Ouvrir le document de test
- [ ] Vérifier la connexion
- [ ] Préparer les liens ou DOI utiles
- [ ] Garder le déroulé horaire visible

## Recommandation pratique

Pour rester prudent :
1. utiliser sans hésiter le logo Wikimedia Commons ;
2. utiliser les captures officielles Zotero avec mention de source dans une petite note ou en fin de support ;
3. éviter pour l'instant les captures App Store iOS, moins claires juridiquement.
